# 🔍 Forensic Hunter

A lightweight Android forensic toolkit built using Termux and ADB to collect, analyze, and generate reports from Android devices.

---

## 🚀 Features

- 📱 Device Information Extraction  
- 📦 Installed Apps Listing  
- 📂 File Collection (Download, DCIM)  
- 📝 Case-Based Report Generation  
- 📡 WiFi ADB Support  
- 🧩 Modular Script Architecture  
- ⚡ Fast & Lightweight (Termux-based)  

---

## 🛠️ Requirements

- Termux
- ADB (android-tools)
